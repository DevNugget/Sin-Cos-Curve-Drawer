# Sin/Cos Curve Drawer
Draws sin/cos curve in the terminal.

## Usage
**Options**:
`sin`, `cos`

```
python wave.py <option>
```

**Arguments**
```
python wave.py <option> -arg <number> <number>
```

**Example**
```
python wave.py sin -arg 2 1
```

The above command draws a sin curve of:  
`sin2X^1`
